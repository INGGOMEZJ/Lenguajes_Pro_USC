Unidad 2 - Patrones de Diseño

Patrones de Diseño aprendidos en la Unidad 2

En esta unidad, se exploraron diferentes patrones de diseño, incluido el Patrón Estructural Adapter y el Patrón de Comportamiento Strategy.

Patrón Estructural Adapter

                
                  class Adaptee:
                      # ... (código del Adaptee)

                  class Client:
                      # ... (código del Client)

                  class Adapter(Client):
                      # ... (código del Adapter)

                  # Ejemplo de implementación del adapter:
                  Pdf_reader = Client()
                  Word_reader = Adaptee()

                  print(Adapter(Word_reader).request)

Patrón de Comportamiento Strategy

                
                  class Strategy:
                      # ... (código del Strategy)

                  class ConcreteStrategyA(Strategy):
                      # ... (código del ConcreteStrategyA)

                  class ConcreteStrategyB(Strategy):
                      # ... (código del ConcreteStrategyB)

                  class Context:
                      # ... (código del Context)

                  # Ejemplo de uso del patrón strategy (general):

                  result_str1 = contexto_str1.execute_strategy()

                  strategyB = ConcreteStrategyB()
